# 🚀 Adventure RPG
<img src="Adventure RPG BG.png"/>
Unity 3D RPG 전투 게임

## 🎮 게임 특징
- 👾 적 우주선과 탄막 회피, 타격 메커니즘
- 💥 스페이스바로 연속 발사되는 탄막 시스템
- 🛰️ 좌우 방향키로 플레이어 우주선 조작
- 🌌 우주 배경 속 몰입감 있는 전투 연출
- 🔊 박진감 넘치는 배경음악과 타격 사운드

## 📸 플레이 영상

## 🎮 조작법
- **이동**: 마우스 좌클릭
- **공격**: 적 좌클릭
- **세이브**: S
- **세이브 삭제**: Delete
- **세이브 불러오기** : L
